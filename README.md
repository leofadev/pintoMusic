# Pinto Music

Aplicación Ionic/Angular preparada para GitHub Pages.

https://leofadev.github.io/pintoMusic/

Rutas útiles dentro de la app:

- https://leofadev.github.io/pintoMusic/#/intro

- https://leofadev.github.io/pintoMusic/#/menu/home

## Desarrollo local

```bash
npm install
npm start
```

## Build para GitHub Pages

```bash
npm run build:pages
```

El proyecto usa rutas con hash para que funcione correctamente en un hosting estático como GitHub Pages.
